# Figma + Canva Handoff Prompt: Packsmith

Design assets for a Packsmith template pack.

Goal:

Create visual previews that make the template feel real, useful, and worth trying before purchase.

Required assets:

- Marketplace cover image
- Mobile preview screen
- Notion/workspace dashboard preview
- Instagram carousel
- Short launch deck
- Feature preview cards

Creative direction:

- Retro-modern product feel
- Clear buyer promise
- Real workflow preview sections
- Strong but readable contrast
- Use visual hierarchy instead of long paragraphs
- Show the actual template experience, not generic abstract cards

Safety:

- Do not include private data, secrets, fake testimonials, or guaranteed earnings claims.
- For healthcare, show operations and marketing workflows only.
