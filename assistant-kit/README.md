# Packsmith Assistant Kit

Packsmith can be used as a web app and as a lightweight assistant workflow. This kit gives users copy-ready instructions for ChatGPT, Claude, Figma, and Canva so they can validate the template-pack idea before a full commercial integration is ready.

## Use Cases

- Turn a rough template idea into a sellable pack brief.
- Create Notion, Canva, Figma, Instagram, Gumroad, LinkedIn, and X launch assets.
- Keep the workflow friendly for non-technical creators.
- Use INR pricing and validation-first launch steps.

## Files

- `chatgpt-packsmith-instructions.md`: paste into ChatGPT custom instructions or a custom GPT-style project.
- `claude-packsmith-project.md`: paste into Claude project instructions.
- `figma-canva-handoff.md`: paste into Figma/Canva generation or design handoff workflows.

## Security Rules

- Do not paste API keys, OAuth secrets, Notion tokens, or private customer data into prompts.
- Review all claims before publishing.
- Avoid fake testimonials, guaranteed income claims, medical advice, and legal advice.
- Use healthcare templates only for operations, marketing workflow, appointment workflow, lead tracking, and reputation support.
